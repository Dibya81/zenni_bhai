## How to Use

1. **Double-click `index.html`** to open in any modern browser
2. Navigate using the top navigation bar or bottom buttons
3. Click any house in the birth chart SVG to learn about it
4. Click any planet row to see detailed analysis
5. Click Dasha entries to expand them
6. Visit `health.html` for the body/health/relationship analysis
7. Visit `summary.html` for a printable overview — use **Print / Save PDF** button to export
